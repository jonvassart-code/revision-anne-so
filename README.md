# 🇧🇪 Fiches de Révision — Système Politique et Administratif Belge

**App interactive de révision** pour le cours de Système Politique et Administratif Belge (SPAB) — UCLouvain, Bac 3 ECGE.

## 📖 Contenu

11 chapitres avec synthèses, chronologies, lexiques et onglets spécialisés.

## 🚀 Déploiement

Le site est déployé automatiquement via [Vercel](https://vercel.com) à chaque push sur `main`.

## 🛠️ Stack

- React 18
- Vite
- CSS-in-JS (inline styles)
