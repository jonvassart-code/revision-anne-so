# 🇧🇪 Fiches de Révision — Système Politique et Administratif Belge
 
**App interactive de révision** pour le cours de Système Politique et Administratif Belge (SPAB) — UCLouvain, Bac 3 ECGE.

11 chapitres couverts avec synthèses, chronologies, lexiques et onglets spécialisés :
 
| # | Chapitre | Onglets spéciaux |
|---|----------|-----------------|
| 1 | La Belgique | ⏳ Chrono |
| 2 | Les Clivages | ⚔️ Clivages · 🏗️ Réformes · 🔐 Mécanismes |
| 3 | Les Partis | 🗳️ Familles politiques |
| 4 | Consociationalisme | ⏳ Pactes |
| 5 | Gouvernements & Parlements | 🏛️ Parlements |
| 6 | Système Électoral | ⏳ Droit de vote |
| 7 | Innovations Démocratiques | — |
| 8 | Système Administratif | — |
| 9 | Enjeux Contemporains | ⚖️ Justice |
| 10 | Particratie & Cabinets | — |
| 11 | L'Extrême Droite | ⏳ Phases · 🇧🇪 Wallonie |
 
Chaque chapitre inclut **📖 Synthèse** (sections dépliables) et **🧠 Lexique**.
 
## 🚀 Utilisation
 
Le fichier `SPAB.jsx` est un composant React autonome. Pour le visualiser :
 
1. Colle le contenu dans [Claude.ai](https://claude.ai) comme artifact React
2. Ou intègre-le dans un projet React/Vite existant
## 🛠️ Stack
 
- React (Hooks)
- CSS-in-JS (inline styles)
- Aucune dépendance externe
